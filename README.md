## Bring the cozy vibes and natural abundance of island life to Satisfactory!

The Island Living mod fully populates the Island in the southwest corner of the game map, providing every resource the game has to offer in a peaceful setting with no harmful creatures. You can start a new game and complete it without ever leaving your island sanctuary! Or, add a slice of island life to an existing save to use as your own peaceful retreat or factory of factories!

## Features:

- ### New Resource Nodes
  Adds __34 new pure resource nodes__ to the Island - including all of the games resources:
  - 6 x Iron
  - 4 x Copper
  - 4 x Limestone
  - 4 x Coal
  - 3 x Oil
  - 2 x Quartz
  - 2 x Caterium
  - 2 x Bauxite
  - 2 x Sulfur
  - 2 x SAM
  - 2 x Nitrogen Wells (8 satellites each)
  - 1 x Uranium Node, isolated on the northern rock away from the main island

- ### New Geysers

  Adds 4 __pure geysers__ (in 2 clusters). *Can be disabled in mod options (see below).*


- ### New Plant Spawns

  Adds all types of plants to the Island - including __Paleberries__, __Beryl Nuts__, __Bacon Shrooms__, and __Mycelia__. 
  - __Re-spawn Timer Adjustment:__ The re-spawn time has been reduced on these island resources to 10 minutes (or on any reload of the save). 
  - __Permanent Removal:__ You can also permanently remove them with your chainsaw, and doing so will reward you with a much larger stack of that resource.

- ### New Mercer Sphere locations

  Adds __15 new Mercer Sphere spawns__ across the island. Each of these special island spheres grants you __10 Mercer Spheres on pickup__ - providing a total of 150 spheres across the island. About half the base game total, this allows you to fully unlock all MAM nodes and construct about 50 dimensional depots.

- ### New Somersloop Locations

  Adds __10 new Somersloop spawns__ across the island. Each of these special island sloops grants you __5 Somersloops on pickup__ - providing a total of 50 spheres across the island. About half the base game total, this allows you to fully unlock all MAM nodes, construct a few power augmenters, and/or sloop several production lines.

- ### New Powerslug Locations

  Adds __60 new Powerslug spawns__ across the island (20 of each color). Each of these special island slugs grants you __3 Powerslugs on pickup__ - providing a total of 60 slugs per color across the island. 

- ### New Mini-Drop Pod Locations

  Adds __15 new mini drop pods__ across the island. Each of these small drop pods __contains 7 Hard Drives__, providing a total of 105 hard drives across the island. As in the base game, each pod has helpful debris scattered around it *(can be disabled, see mod options below).*

- ### New Creature Spawns

  The island would hardly be living without creatures! The island is now populated with all the games types of non-harmful creatures, including:
   - 15 __Lizzard Doggos__ ready for you to tame
   - Many non-flying __birds__ (those little green guys that scream at the sky and block your foundations)
   - Several __Beans__ (Giraffe-Whale-Tick-etc)
   - Several __Beetles__ (the poor guy that gets run over on the 1.2 login screen)
   - A new __Manta__, cruising the skies above the Island.
 
  All of these creature types can be individually disabled or enabled from the mods options _(see details on the mod options below)._

- ### New Creature Remains

  As you may know, FICSIT requires pioneers research local hazardous fauna. These rude beasts have no place on this peaceful island, but luckily they used to! Scattered around the island you will find the remains of former:
  - Hogs 
  - Spitters
  - Stingers
  - Hatchers. 
  
  It may take some keen searching, but a total of 50 of each type (in piles of 10) are available.


- ### Mod Options:

  To ensure you can configure your personal best island life, several features can be enabled or disabled. Note that all of these settings must be changed on the main menu as they require the game to reload. You can find these options on the menu under Mods>IslandLiving.

  - Creatures:
    - __Enable Beans__: When enabled, includes Bean creatures on the Island. Default: Enabled.
    - __Enable Birds__: When enabled, includes Bird creatures on the Island. Default: Enabled.
    - __Enable Manta__: When enabled, includes a flying Manta on the Island. Default: Enabled.
    - __Enable Beetles__: When enabled, includes Beetle creatures on the Island. Default: Enabled.
    - __Enable Doggos__: When enabled, includes Lizzard Doggo creatures on the Island. Default: Enabled. <font color="red">WARNING: If you disable this option, any island Doggo's you have tamed will vanish. If you later re-enable this, you will have to retame them. </font>
  - Other:
    - __Enable Geysers__: When enabled, incldues geyers on the island. Default: Enabled. 
    - __Enable Dropped Items__: When enabled, includes dropped items at island drop pod sites. Default: Enabled.
<br><br>
### Game Mode Recommendation
  - While certainly not required, I recommend pairing this mod with the new 1.2 Game Mode setting to reduce Recipe Cost. This mod was heavily inspired by this new feature and it's ability to make the game much more compact when set to 0.5 or 0.25. That said, you can absolutely complete the game on normal settings with this mod without ever leaving the island - you may just have to invest a bit more in verticality =).

<font color="yellow">

### Warnings! </font>
  - <font color="yellow">Mod Soft-Conflict: __Paradise Island Enhanced:__</font>
    __Island Living__ and __Paradise Island Enhanced__ both spawn many things on this area of the map. I have tested to ensure that having both mods will not cause a hard game crash. However, you will find many nodes overlap and interrupt, and some may become non-functional. You should only use one of these two mods. Paradise Island Enhanced is significantly more balanced to the base game-play loop and is an excellent choice if you wish to start on the island but be required to branch out in order to complete the game.
  - <font color="yellow">Multiplayer and Dedicated Server: Untested</font>
    __Island Living__ has not been tested on multiplayer or on dedicated servers. Use in those configurations at your own risk. If you do and encounter bugs or want to report feedback, please contact me using the contact details lower on this page.
